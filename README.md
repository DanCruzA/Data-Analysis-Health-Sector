# Data-Analysis-Health-Sector

Proyecto: Análisis de Operaciones Clínicas

Objetivo: Optimizar la toma de decisiones operativas mediante el análisis de citas médicas.

Stack Tecnológico:

Python (Pandas): Para la limpieza de datos (manejo de fechas inconsistentes y valores nulos).

Matplotlib: Para la visualización de la demanda por especialidad.

Jupyter Notebook: Entorno de desarrollo interactivo.

Metodología: Se simuló un proceso ETL donde se ingesta data cruda con errores típicos (formatos de fecha mixtos, datos faltantes), se transforma para asegurar la calidad del dato y se visualizan los resultados para identificar las especialidades con mayor tráfico.
